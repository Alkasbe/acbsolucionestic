Convertidor JSON ↔ Excel

Resolución 2275 de 2023 FEV-RIPS


Instrucciones

1. Selecciona un archivo JSON o Excel usando el botón arriba.

2. Presiona "Convertir a Excel" para transformar un JSON en Excel.

3. Presiona "Convertir a JSON" para transformar un Excel en JSON.

4. El archivo convertido se descargará automáticamente.
