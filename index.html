Convertidor JSON ↔ Excel

Resolución 948 de 2026 FEV-RIPS

Instrucciones de uso

1. Selecciona un archivo JSON o Excel usando el botón de arriba.

2. Presiona "Convertir a Excel" para transformar un JSON en Excel.

3. Presiona "Convertir a JSON" para transformar un Excel en JSON.

4. El archivo convertido se descargará automáticamente en tu dispositivo.
